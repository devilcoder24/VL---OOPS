Operators in Java - I

Java operators are special symbols that perform operations on variables and values. They are used to manipulate data and variables in expressions.In this lab you will be studying about the various types of operators.

Note: Before moving to the types of operators, it's important to know that how we can take input from the users in Java. So, it's possible by using the Scanner class.

Scanner Class

The Scanner class is part of the java.util package. It is used to take input from different sources, i.e., to accept user-choice input from keyboard (and other sources). It can be used by importing the class using following syntax:

import java.util.Scanner;

Program:

import java.util.Scanner;

class Test
{
    public static void main(String args[])
    {
        Scanner sc = new Scanner(System.in);    //for taking input 
        
        System.out.print("Enter your age: ");   
        int age = sc.nextInt();                 //reads a new integer
        
        System.out.print("Age: " + age);
    }
}

Output:

Enter your age: 24 (user's-choice)
Age: 24

Types of Operators

They are of the following types :

Arithmetic Operators

Arithmetic operators are used to perform basic mathematical operations.

  • Addition(+) : It is used to perform the addition of two numbers.
  • Subtraction(-) : It is used to perfoem the subtraction of two numbers.
  • Multiplication(*) : It is used to perform the multiplication of two numbers.
  • Division(/) : It is used to divide the first number by the second and return the quotient as output.
  • Modulo(%) : It is used to divide the first number by the second and return the remainder as output.

Program:

class Test
{
    public static void main(String args[])
    {
        int a=10;
        int b=5;
        System.out.println("Addition: " + (a + b));      
        System.out.println("Subtraction: " + (a - b));  
        System.out.println("Multiplication: " + (a * b));
        System.out.println("Division: " + (a / b)); 
        System.out.println("Modulus: " + (a % b));
    }
}

Output:

Addition: 15
Subtraction: 5
Multiplication: 50
Division: 2
Modulus: 0

Relational Operators

Relational operators are used to compare two values.

  • Equal to(==) : It compares two values and returns true if equal, otherwise false.
  • Not equal to(!=) : It compares two values and returns false if equal, otherwise true.
  • Greater than(>) : It checks if first value is greater than second then returns true, otherwise false.
  • Less than(<) : It checks if first value is less than second then returns true, otherwise false.
  • Greater than or equal to(>=) : It checks if the first value is greater than or equal to the second then returns true, otherwise false.
  • Less than or equal to(<=) : It checks if the first value is less then or equal to the second then returns true, otherwise false.

Program:

class Test
{
    public static void main(String args[])
    {
        int a=20;
        int b=10;
        System.out.println(a == b); 
        System.out.println(a != b);
        System.out.println(a > b); 
        System.out.println(a < b);
        System.out.println(a >= b);
        System.out.println(a <= b);
    }
}

Output:

false
true
true
false
true
false

Logical Operators

Logical operators are used to combine multiple boolean expressions.

  • Logical AND(&&) : This operator returns true only if both conditions are true. If either condition is false, then result is false.
  • Logical OR(||): This operator returns true if at least one condition is true. It returns false only when both conditions are false.
  • Logical NOT(!): It reverses the logical value. If a condition is true, then it becomes false, and if a condition is false then it becomes true.

Program:

class Test
{
    public static void main(String args[])
    {
        boolean a = true;
        boolean b = false;
        System.out.println(a && b);
        System.out.println(a || b); 
        System.out.println(!a);     
    }
}

Output:

false
true
false

Assignment Operators

Assignment operators are used to assign values to variables.

  • Simple assignment(=) : It is used to assign a vlue to a variable.
  • Add and assign(+=) : It adds the right-hand value to the variable and assigns the result back.
  • Subtract and assign(-=) : It subtracts the right-hand value from the variable and assigns the result back.
  • Multiply and assign(*=) : It multiplies the variable by the right-hand value and assigns the result back.
  • Divide and assign(/=) : It divides the variable by the right-hand value and assigns the result back.
  • Modulo and assign(%=) : It stores the remainder of division back into the variable

Program:

class Test
{
    public static void main(String args[])
    {
        int a = 10;
        a += 5;  
        System.out.println(a);
        a -= 5;  
        System.out.println(a);
        a *= 5;  
        System.out.println(a);
        a /= 5;  
        System.out.println(a);
        a %= 5;  
        System.out.println(a);
    }
}

Output:

15
10
50
50
0
