Operators in Java - II

Java operators are special symbols that perform operations on variables and values. They are used to manipulate data and variables in expressions.In this lab you will be studying about the some more types of operators.

Types of Operators

They are of the following types :

Unary Operators

Unary operators operate on a single operand.

  • Unary Plus(+) : It indicates a positive value (doesn‚Äôt change the number).
  • Unary Minus(-) : It negates the value (makes positive ‚Üí negative, negative ‚Üí positive).
  • Increment(++) : It increases the value of a variable by 1.
    • Pre-increment(++operand) : First increases, then use the value.
    • Post-increment(operand++) : First uses, then increases the value.
  • Decrement(--) : It decreases the value of a variable by 1.
    • Pre-decrement(--operand) : First decreases, then uses the value.
    • Post-decrement(operand--) : First uses, then decreases the value.

Program:

class Test
{
    public static void main(String args[])
    {
       int a = 10;
       int result = +a;
        System.out.println("The value of result after unary plus is: " + result);
       
       result = -a;
       System.out.println("The value of result after unary minus is: " + result);
       
       int b = 20;
       result = b++;
       System.out.println("The value of result after post-increment is: " + result);

       result = ++b;
       System.out.println("The value of result after pre-increment is: " + result);
       
       int c = 30;
       result = c--;
       System.out.println("The value of result after post-decrement is: " + result);

       result = --c;
       System.out.println("The value of result after pre-decrement is: " + result);
    }
}

Output:

The value of result after unary plus is: 10
The value of result after unary minus is: -10
The value of result after post-increment is: 20
The value of result after pre-increment is: 22
The value of result after post-decrement is: 30
The value of result after pre-decrement is: 28

Bitwise Operators

Bitwise operators perform operations on bits.

  • Bitwise And (&) : It compares each bit of two numbers, and the result is 1 only if both bits are 1, else 0.
  • Bitwise OR (|) : It compares each bit of two numbers, and the result is 1 if at least one bit is 1, else 0.
  • Bitwise XOR (^) : Exclusive OR, and the result is 1 if bits are different, 0 if they are the same.
  • Bitwise NOT (~) : It flips all bits (1 becomes 0, 0 becomes 1). In Java, numbers are stored in two‚Äôs complement, so ~a = -(a+1).

Program:

class Test
{
    public static void main(String args[])
    {
        int a = 5;
        int b = 7;
        System.out.println("a&b = " + (a & b));
        System.out.println("a|b = " + (a | b));
        System.out.println("a^b = " + (a ^ b));
        System.out.println("~a = " + ~a);

    }
}

Output:

a&b = 5
a|b = 7
a^b = 2
~a = -6

Bitwise-Shift Operators

Shift operators are used to shift the bits of a number left or right, thereby multiplying or dividing the number by two, respectively. They can be used when we have to multiply or divide a number by two.

  • Left shift (<<) : This operator shifts all bits of a number to the left by a specified number of positions. In other words, each left shift = multiply the number by 2^n.
  • Right shift (>>) : This operator shifts all bits to the right by a specified number of positions. In other words, each right shift = divide the number by 2^n. It also keeps the sign bit (arithmetic shift).
  • Unsigned right shift (>>>) : This operator shifts all bits to the right, but does not keep the sign bit. It fills leftmost bits with 0. It is useful for unsigned values (though Java int is always signed).

Program:

class Test
{
    public static void main(String args[])
    {
        int a = 4;
        System.out.println("Bitwise Left Shift: " + (a << 2));
        System.out.println("Bitwise Right Shift: " + (a >> 2));
        System.out.println("Bitwise Unsigned Right Shift: " + (a >>> 2));
    }
}

Output:

Bitwise Left Shift: 16
Bitwise Right Shift: 1
Bitwise Unsigned Right Shift: 1

Ternary Operator

The ternary operator is a shorthand form of the if-else statement.

Syntax:

result = (condition) ? value_if_true : value_if_false;
  • If condition is true, then it returns value_if_true.
    • If condition is false, then it returns value_if_false.

Program:

class Test
{
    public static void main(String args[])
    {
        int a = 10;
        int b = 5;
        int max = (a > b) ? a : b;
        System.out.println("The maximum between two numbers is: " +max);
    }
}

Output:

The maximum between two numbers is: 10
